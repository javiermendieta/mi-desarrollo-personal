# Mi Desarrollo Personal 🚀

Aplicación web completa para desarrollo personal con múltiples módulos integrados.

## 📱 Módulos

- **📊 Dashboard** - Vista general con estadísticas y resumen
- **📅 Calendario** - Vista día/semana/mes con drag & drop
- **💪 Deportes** - Gestión de deportes, rutinas y ejercicios
- **🧘 Yoga & Meditación** - Timer con alarma para meditar
- **📚 Lectura** - Control de libros, páginas y notas
- **📓 Diario** - Entradas diarias + Creencias Limitantes
- **🎯 Metas** - Objetivos con progreso
- **✅ Hábitos** - Check-in diario con rachas
- **💰 Finanzas** - Ingresos, gastos y ahorros
- **❤️ Salud** - Hidratación y sueño
- **📝 Notas** - Notas rápidas con colores
- **🤖 Agente AI** - Asistente personal con memoria

## 🛠️ Tecnologías

- **Next.js 15** - Framework React
- **TypeScript** - Tipado estático
- **Tailwind CSS** - Estilos
- **shadcn/ui** - Componentes UI
- **Supabase** - Base de datos
- **Zustand** - Estado global

## 🚀 Deploy en Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/TU-USUARIO/mi-desarrollo-personal)

## ⚙️ Variables de Entorno

Crea un archivo `.env.local` con:

```env
NEXT_PUBLIC_SUPABASE_URL=tu-url-de-supabase
NEXT_PUBLIC_SUPABASE_ANON_KEY=tu-clave-anonima
```

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/TU-USUARIO/mi-desarrollo-personal.git

# Instalar dependencias
bun install

# Ejecutar en desarrollo
bun run dev
```
